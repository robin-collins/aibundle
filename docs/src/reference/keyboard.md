# Keyboard Shortcuts Reference 🎹

## Navigation Keys 🚀

| Key | Action | Context |
|-----|--------|---------|
| ↑/↓ | Move selection | Always |
| PgUp/PgDn | Move by 10 items | Always |
| Enter | Open directory | On directory |
| Backspace | Parent directory | Always |
| Tab | Expand/collapse | On directory |

## Selection Keys ✨

| Key | Action | Context |
|-----|--------|---------|
| Space | Toggle selection | Always |
| * | Toggle all visible | Always |
| / | Start search | Always |
| ESC | Clear search/modal | Search/Modal |

## Operation Keys 🛠️

| Key | Action | Context |
|-----|--------|---------|
| c | Copy to clipboard | With selection |
| f | Toggle format | Always |
| n | Toggle line numbers | Not JSON format |
| s | Save configuration | Always |
| q | Quit application | Always |

## Filter Keys 🔍

| Key | Action | Context |
|-----|--------|---------|
| i | Toggle default ignores | Always |
| g | Toggle .gitignore | Always |
| b | Toggle binary files | Always |

## Help & Information ℹ️

| Key | Action | Context |
|-----|--------|---------|
| h | Show help | Always |
| ? | Show help | Always |

## Modal Navigation 📝

| Key | Action | Context |
|-----|--------|---------|
| ESC | Close modal | Modal open |
| Any key | Close help | Help modal |
| PgUp/PgDn | Scroll help | Help modal |

## Status Indicators 📊

### Format Status
- XML: Default structured format
- MD: Markdown documentation format
- JSON: Machine-readable format

### Toggle States
- [x] Feature enabled
- [ ] Feature disabled
- [>] Current selection

## Best Practices 💡

1. Learn Essential Keys
   - Navigation: ↑/↓, Enter
   - Selection: Space, *
   - Operations: c, f, q

2. Use Efficient Combinations
   - Search then select
   - Expand then bulk select
   - Format before copy

3. Watch Status Bar
   - Current settings
   - Selection count
   - Operation feedback

4. Use Help When Needed
   - 'h' for quick reference
   - Check status bar
   - Review modal messages
