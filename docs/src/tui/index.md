# Terminal User Interface (TUI) 🖥️

AIBundle's Terminal User Interface provides an intuitive, interactive way to select and bundle files.

## Key Components 🔑

### File Browser
- 📂 Hierarchical file view
- 🔍 Real-time search
- ✨ Visual file type indicators
- 📌 Selection checkboxes

### Status Bar
- 📊 Selection count
- ⚙️ Current settings
- 🎮 Available commands
- 📝 Search status

### Modal Dialogs
- ℹ️ Help information
- 📋 Copy confirmation
- ⚠️ Warning messages
- 🔄 Operation status

## Navigation 🚀

| Key | Action |
|-----|--------|
| ↑/↓ | Move selection |
| Enter | Open directory |
| Tab | Expand/collapse folder |
| Space | Select/deselect item |
| / | Search mode |

- 🎯 Visual file selection
