# Introduction to AIBundle 📦

AIBundle is a powerful Terminal User Interface (TUI) and Command Line Interface (CLI) tool designed to help developers efficiently bundle code files for AI and LLM interactions. Whether you're working with ChatGPT, Claude, or other AI assistants, AIBundle makes it easy to prepare and format your code for productive discussions.

## What is AIBundle? 🤔

AIBundle serves as a bridge between your codebase and AI tools by:
- 🎯 Selecting specific files and directories
- 🔄 Converting them into AI-friendly formats
- 📋 Copying them directly to your clipboard
- 🎨 Supporting multiple output formats (XML, Markdown, JSON)

## Key Features 🌟

### Terminal User Interface (TUI)
- 📂 Visual file browser with folder expansion
- ✨ Intuitive keyboard controls
- 🔍 Real-time search functionality
- ⚡ Quick file selection and copying

### Command Line Interface (CLI)
- 🚀 Quick file bundling with pattern matching
- 💾 Direct file output or clipboard copying
- 🎮 Flexible format options
- 🔧 Powerful filtering capabilities

### Smart Filtering
- 🎯 .gitignore support
- 🚫 Default ignore patterns for common directories
- ⚙️ Configurable ignore rules
- 📊 Binary file handling

## Why AIBundle? 💡

When working with AI assistants, sharing code context is crucial but can be cumbersome. AIBundle solves this by:
1. Streamlining the selection process
2. Maintaining consistent formatting
3. Respecting project structure
4. Preserving file relationships
